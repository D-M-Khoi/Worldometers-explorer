# Getting started
Prerequisites:
* Python 3.8+
* Numpy
* Matplotlib
* Pandas
* Selenium 3
* Chrome and chromedriver
# Usage
* Crawling: Run all cells in ```crawler.ipynb```
* Visualization Run all cells in ```visualize.ipynb```
# Exploratory data analysis
